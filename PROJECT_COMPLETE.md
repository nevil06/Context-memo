## Commands Available

### Core Commands (v1.x)
```bash
memo init                    # Initialize .recall/ folder
memo scan                    # Build AST registry + graph
memo scan --local            # Local-only (no API)
memo scan --quick            # Faster scan
memo load                    # Load memory to clipboard
```
