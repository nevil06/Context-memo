COMMAND FLOW:

1. recall init
   └─> Creates .recall/ folder with templates

2. recall config --key KEY
   └─> Saves API key to ~/.recall/config.json

3. recall scan [--quick]
   ├─> scanner.js: Walks project, categorizes files
   ├─> prompt.js: Builds Gemini prompt
   ├─> gemini.js: Calls API
   └─> Saves to .recall/memory.yaml

4. recall load [--mode]
   ├─> Reads .recall/memory.yaml
   ├─> assembler.js: Builds briefing
   ├─> clipboard.js: Copies to clipboard
   └─> Prints to terminal

5. recall status
   ├─> Reads .recall/memory.yaml
   └─> Displays terminal dashboard

6. recall update [message]
   ├─> Interactive prompts
   ├─> Updates .recall/task_state.yaml
   └─> Syncs to .recall/memory.yaml

7. recall install <agent>
   └─> Creates agent-specific config file
